How to export a clean PNG

  1. Open this file in Chrome (drag-drop into a tab, or file:// path).
  2. Open DevTools: ⌘ + ⌥ + I
  3. Open Command palette: ⌘ + ⇧ + P
  4. Type "Capture node screenshot" — it appears in the dropdown.
  5. In the Elements panel, click the <div class="banner crexi"> (or linkedin) so it's selected, then run the command.
  6. Chrome saves a pixel-perfect PNG at the exact div dimensions — 2000×500 or 1584×396 — regardless of your screen size.
